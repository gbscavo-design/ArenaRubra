# F9P1a – Storage Import/Export UX Hotfix

Base: C2-STABLE-1-F9P1-APK-M4c.

Correzione UX del Deck Builder:

- `Scarica tutti deck .json` genera un file JSON scaricabile con tutti i deck salvati.
- `Copia tutti deck JSON` copia lo stesso payload negli appunti.
- `Importa da file .json` apre il selettore file e legge il JSON selezionato.
- `Incolla JSON` apre una textarea: va incollato il testo JSON integrale, non un percorso o URL.
- Feedback utente esplicito su import/export riuscito o fallito.

Nessuna modifica a gameplay, AI, deck runtime, mappa, stat/unità, tattiche o regola no-overflow.
